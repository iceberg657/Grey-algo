# Session Filter
The Session Filter allows you to view detailed market analysis tailored to specific trading sessions (London, New York, Asian).

## Features
- **Economic Events**: Major events happening within the session.
- **Asset Analysis**: Affected pairs, volatility, and bullish/bearish trends.
- **Trading Recommendations**: Desired assets to trade.
- **Automatic Refresh**: Updates every hour and at session changes.

## How to use
1. Select your desired trading session from the tabs.
2. View the detailed analysis generated by Gemini.
3. The data refreshes automatically to ensure you have the latest insights.
